# 🔐 Anomaly-Based Intrusion Detection System

A real-time Intrusion Detection System (IDS) that uses machine learning (Isolation Forest) to detect anomalies in network traffic.

---

## 📌 Overview
This project captures live network packets and analyzes them using a trained machine learning model to identify suspicious or abnormal behavior. It helps detect unknown cyber-attacks by learning normal traffic patterns.

---

## 🚀 Features
- Real-time packet capture using Scapy  
- Machine learning-based anomaly detection  
- Detection of unknown attacks  
- Severity-based alerts (Low / Medium / High)  
- GUI dashboard with live logs and visualization  
- Sound alerts for intrusions  
- Log storage and export functionality  

---

## 🛠 Technologies Used
- Python  
- Scapy  
- Scikit-learn  
- Tkinter  
- Matplotlib  
- NumPy, Joblib
